# Quick Reference Guide for Tap-A-Beat

## Basic Controls
*   **Turn Knob**: Change BPM (Speed).
*   **Click**: Start / Stop.
*   **Push & Turn (Held)**: Adjust Volume.
*   **Double Click**: Open Quick Menu (Time Signature, Subdivisions, Load Preset).
*   **Long Press (> 1s)**: Open Main Menu (Trainer, Tuner, Settings).

## Main Menu Overview
1.  **Metric**: Set Time Signature (e.g. 4/4, 3/4).
2.  **Subdiv**: Add clicks between beats (Eighths, Triplets).
3.  **Taptronic**: Tap rhythm on case to set speed.
4.  **Trainer**: Auto-speed-up mode for practice.
5.  **Timer**: Set practice alarm (1-60 mins).
6.  **Tuner**: Tune your instrument (A4 reference adjustable).
7.  **Presets**: Save/Load your settings.

## Special Functions

### Volume & Silent Mode
*   Turn volume all the way down to 0 to enter **Silent Mode**.
*   In Silent Mode, only the LED and Vibration (Haptic) work.

### Saving a Preset
1.  Go to **Menu -> Presets -> Save**.
2.  Turn to choose a slot (1-50).
3.  **Click** to save instantly.
4.  **Hold Click** to assign the preset to a specific Setlist.

### Tempo Trainer
1.  Go to **Menu -> Trainer**.
2.  Set **Start BPM**, **End BPM** (Target), and **Step** (Increase amount).
3.  Select **Start Trainer** at the bottom.
4.  The BPM will automatically increase after the set number of bars.

### Practice Timer
1.  Go to **Menu -> Timer**.
2.  Set Duration (Minutes).
3.  Click **Running/Stopped** to toggle.
4.  An alarm will sound when time is up.

## Visual Feedback Colors
*   **Red Flash**: Accent (First beat of bar).
*   **Blue Flash**: Normal beat.
